# 内容工厂引擎 — 原生整合模块

来源融合: MoneyPrinterTurbo(#23) + autoresearch(#26) + Deep-Live-Cam(#21)

## MoneyPrinterTurbo — 自动化内容生产

```
一条指令 → 完整视频/图文

输入: "AI创业2026年趋势分析"
  │
  ├─ 1. 素材采集
  │    ├─ YouTube 相关视频
  │    ├─ Google/Bing 搜索
  │    └─ RSS 订阅源
  │
  ├─ 2. 内容生成
  │    ├─ LLM 撰写脚本
  │    ├─ 拆分为分镜
  │    └─ 生成字幕
  │
  ├─ 3. 视觉制作
  │    ├─ AI 生成配图/素材
  │    ├─ 自动匹配视频片段
  │    └─ 合成转场效果
  │
  ├─ 4. 音频处理
  │    ├─ TTS 语音合成
  │    ├─ BGM 自动匹配
  │    └─ 音量均衡
  │
  └─ 5. 输出
       ├─ MP4 视频
       ├─ Markdown 图文
       └─ 多平台分发
```

## autoresearch — 自主调研引擎

```
调研循环:
  1. 接收研究主题
  2. 自动搜索多个来源
  3. 阅读并提取关键信息
  4. 交叉验证（至少2个独立来源）
  5. 发现新线索 → 扩展搜索
  6. 信息饱和 → 停止
  7. 生成结构化报告

饱和判断:
  连续3轮搜索无新信息 → 信息饱和 → 停止调研
```

## 批量内容生产流水线（OpenGod 优化版）

```
适合 wanzhan.fun 日产3000篇场景:

Phase 1: 关键词挖掘
  源: 百度/Google 长尾词 + 5118 + 爱站
  输出: 每日100个高价值关键词

Phase 2: 模板匹配
  文章类型:
    ├─ 教程型 (how-to)
    ├─ 盘点型 (top-list)
    ├─ 对比型 (vs)
    ├─ 深度型 (ultimate guide)
    └─ 新闻型 (news brief)
  
  每个关键词 → 匹配最佳类型

Phase 3: AI 生成（并行100路）
  Prompt 模板:
    [角色] + [内容类型] + [关键词] + [结构要求] + [风格约束]
  
  质量检查:
    字数 > 800? ✓
    原创度 > 70%? ✓
    可读性 > 60? ✓
    至少3个小标题? ✓

Phase 4: 自动配图
  AI 生成封面图 → CDN 上传 → 插入文章

Phase 5: SEO 发布
  title/description/keywords → CMS 发布 → 百度推送 → sitemap更新
```

## 视频生产模式（Deep-Live-Cam + AI 视频）

```
短视频自动生产线:
  ┌─ 热点采集 (Agent-Reach)
  ├─ 脚本生成 (LLM)
  ├─ 画面生成 (AI 图像/视频)
  ├─ 语音合成 (TTS)
  ├─ 自动剪辑 (timeline编排)
  └─ 批量发布 (多平台)
```

## 在 OpenGod 中的执行方式

内容生产请求：
1. 加载此模块 → 自动匹配最佳生产流水线
2. 小批量试产（10篇） → 质量评估 → 调参
3. 质量稳定 → 全量生产
4. 通过 workflow-engine 编排并行任务
5. 监控指标：收录率/点击率/跳出率/转化率
