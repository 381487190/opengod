## AI Hedge Fund 模拟基金

```
多策略并行:
  价值投资策略    → 长期持有优质标的
  趋势跟踪策略    → 跟随市场方向
  均值回归策略    → 超跌反弹
  事件驱动策略    → 公告/财报/政策
  
资金分配:
  总资金 ¥100,000
  ├─ 价值投资: 40% → 大盘蓝筹
  ├─ 趋势跟踪: 30% → 强势股
  ├─ 均值回归: 20% → 超跌品种
  └─ 事件驱动: 10% → 热点题材

每日复盘:
  当日收益 → 各策略贡献 → 风险指标 → 调仓建议
```

## OpenBB — 专业金融终端

```
OpenBB 数据视图:
  宏观经济 → GDP/CPI/PMI/利率
  行业板块 → 行业轮动/资金流向
  个股深度 → 财报/估值/技术/机构
  全球市场 → A股/港股/美股/外汇/商品
  ETF/基金 → 净值/持仓/折溢价
  另类数据 → 社交媒体/供应链/卫星图像
```

## 数据源整合链

```
数据优先级:
  1. westock-data skill (WorkBuddy 内置，A股/港股/美股全量)
  2. neodata-financial-search (自然语言查询)
  3. wb-finance-skill (统一金融入口)
  4. WebSearch (网络舆情补充)
  5. Agent-Reach 模块 (社媒情绪抓取)
```

## 在 OpenGod 中的执行方式

所有金融分析请求：
1. 优先使用 westock-data / neodata / wb-finance 获取结构化数据
2. 并行启动多策略分析（基本面 + 技术面 + 情绪面）
3. 用河洛太极交叉验证（利/害、快/慢、攻/守）
4. 输出决策仪表盘格式（评分 + 趋势 + 风险 + 操作建议）
5. 通过 Hermes 记录到 learnings/，供历史回测
